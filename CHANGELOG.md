# Changelog

All notable changes to this project will be documented in this file.

The format is based on [Keep a Changelog](https://keepachangelog.com/en/1.0.0/),
and this project adheres to [Semantic Versioning](https://semver.org/spec/v2.0.0.html).

## [3.0.0] - 2026-06-09

### Added
- Initial release of REX-OS v3.0
- Core API service built with Flask
- Lightweight telemetry agent for distributed monitoring
- Rich Terminal UI dashboard for real-time visualization
- RESTful API endpoints for metrics and logs
- Support for multi-node monitoring
- Comprehensive documentation and examples

### Features
- Real-time infrastructure monitoring
- Decoupled architecture for scalability
- Linux/Termux environment support
- JSON-based data transport
- MIT License
